# Testes de Mutação - Projeto Academia Gym

## 1. Introdução
Este documento detalha a aplicação de Testes de Mutação para avaliar a eficácia da suíte de testes atual do projeto.

## 2. Ferramenta Utilizada: `mutmut`
O `mutmut` é uma ferramenta de teste de mutação para Python. Ele aplica mutações no código-fonte e executa a suíte de testes para verificar se as mutações são detectadas.
